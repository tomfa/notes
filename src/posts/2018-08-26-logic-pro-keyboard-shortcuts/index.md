*   Enter: Restart track
*   Space: Play/pause
*   R: Record/toggle record
*   cmd-k: Bring up synthesizer
*   cmd-d: copy track
*   y: bring up library
*   l: loop this
*   , : go back in song
*   . : go forward

Learning tips:

*   Cycle button (shortcut: C) will repeat a marked section
